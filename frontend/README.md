# Lamina-Flow (Frontend)

## Features
1. Workspaces   (global)
2. Workflows    (1:n <=> Workspace:Workflow)
3. Manifests    (global)
4. Manifests    (m:n <=> Workspace:Manifest)
5. Settings     (global)
6. Settings     (1:1 <=> Workspace:Settings)
7. Deployments  (1:m <=> Workspace:Deployment) + Execution Status Collection, etc.
8. Optional Auth
